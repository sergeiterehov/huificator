# Хуификатор JS

Установить-хуевить

```
npm i -s huificator
```

Инициализация-хуяция

```js
// @ts-ignore
// eslint-disable-next-line
if (window.__huificator) window.__huificator();
```

Проще некуда-хуекуда!
